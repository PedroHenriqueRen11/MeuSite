Projetos

FutGol 2D

Jogo 2D em Python que simula um desafio de precisão e reflexos, onde o objetivo é controlar o jogador para interceptar as bolas que caem. Desenvolvido para aplicar conceitos de física e detecção de colisão. O game oferece controles simples, foca na jogabilidade responsiva e proporciona uma experiência desafiadora ao testar a agilidade do jogador.

Jogos Mortais

Protótipo de sistema em Python que explora lógica de controle de fluxo e manipulação de dados em tempo real. Este projeto demonstra a habilidade na organização de código, design de arquitetura de software e aplicação de princípios de POO para criar uma solução eficiente e escalável.

Tratamento de Dados

Pandas desenvolvido em um ambiente Google Colab para análise de dados, otimização de fluxo ou validação de informações. Demonstra capacidade em manipulação de dados, aplicação de lógica condicional e execução de projetos que exigem estruturação e precisão lógica.

Biblioteca Virtual

Sistema de Gerenciamento de Biblioteca desenvolvido em Java. O projeto utiliza princípios de Programação Orientada a Objetos (POO) e programação estruturada para lidar com a inserção, consulta e organização eficiente de acervo físico e digital.

Estrutura de Loja

Trabalho acadêmico em Python que simula um sistema de gestão de loja (estoque/vendas). Demonstra domínio em Estruturas de Dados clássicas (listas, pilhas e filas) e a implementação eficiente de um sistema baseado em Orientação a Objetos (POO).

CiriFood

Cirifood é um protótipo de aplicativo de delivery (Figma). O projeto mapeou e estruturou o fluxo principal do usuário, focando em uma navegação intuitiva e simplicidade. Destaca a habilidade em design centrado no usuário e no uso do Figma para prototipagem de baixa fidelidade.
